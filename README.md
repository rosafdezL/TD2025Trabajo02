# Trabajo 02- Decisión Multicriterio
Este repositorio contiene varios archivos para el segundo trabajo de la asignatura Teoría de la Decisión:
- eleccionciudad.Rmd es un archivo R markdown que contiene la resolución del problema planteado.
- eleccionciudad.html es el archivo HTML del problema.
- eleccionciudad.pdf es el archivo PDF del problema.
- eleccionciudad.ahp es el archivo para realizar el método AHP con la librería ahp.
- tablaahpciudad.png y tablaahpciudad2.png son imágenes correspondientes al método AHP.
- teoriadecision_funciones_multicriterio_utiles.R, teoriadecision_funciones_multicriterio_diagram.R, teoriadecision_funciones_multicriterio.R son los archivos R con las funciones necesarias para la realización del trabajo.
- eleccionciudad.docx es el archivo WORD que contiene páginas webs en las que se ha buscado la información para poder crear la matriz de decisión.

El problema se ha resuelto aplicando todas las técnicas vistas en clase de decisión multicriterio. En este caso, evaluamos seis alternativas con el objetivo de elegir una ciudad para realizar un máster en Estadística, evaluando 8 subcriterios.
